Getting started

  1. Run Setup Sheets from the Setup Wizard menu to create all 14 worksheets.
  2. Open the sheet for the task (e.g. 1. Create Users) and fill in one row per user/group/member.
  3. Run the matching menu item (e.g. Users Manager > 1. Create Users).
  4. Watch the Status column: ✔ means done, ✖ shows the error for that row.

Good to know

  1. Safe to re-run: rows already marked ✔ are skipped, so you can fix failed rows and run the same action again.
  2. Big batches: long runs stop safely before Google’s script time limit — just run the action again to resume.
  3. Rate limits: every API call retries automatically with exponential backoff, so temporary “quota exceeded” errors resolve themselves.
  4. Exports: export sheets are fully rewritten each time you run them.

You must be a Google Workspace super admin (or have the matching admin privileges) for these operations to succeed.

Batchwork is free without limits and 100% user-supported — see ♥ Support Batchwork in the menu.
