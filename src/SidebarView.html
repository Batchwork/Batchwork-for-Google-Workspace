How it works

  1. Pick an operation and click Open sheet.
  2. Fill one row per user, group or member.
  3. Click Run and watch the live progress here.
  4. ✔ / ✖ results land in the sheet's Status column.

Rows marked ✔ are skipped on re-run, so it is always safe to fix errors and run again. Long runs pause safely near Google's time limit — run again to resume.
